申請 APEC 經費

2019 年補助計畫簡介
  1. 申請者：APEC 旗下 21 個經濟體及其公部門、外部利害關係人。
  2. 計畫內容可包含工作坊、研討會、出版品、研究。

關鍵詞中英對照

1. 單位╱職位
  • APEC Secretariat：APEC 祕書處
  • APEC BMC (Budget and Management Committee)：APEC 預算管理委員會
  • APEC Secretariat PMU (Project Management Unit)：APEC 祕書處計畫管理部門
  • PO (Project Overseer)：計畫主持人
  • Program Director：計畫主任
  • SOM (Senior Officials' Meeting)：資深官員會議
2. 其他
  • APEC Project Design Amendment and Extension form：計畫設計修正與延展申請表
  • APEC Project Funding Source：APEC 計畫補助來源
  • APEC Publications Toolkit：APEC 出版工具包
  • Completion Report：完工報告
  • Concept Note：概念文件
  • Guidebook on APEC Project：APEC 計畫指南
  • Letter of Acceptance：獲選通知書
  • Monitoring Report：監控報告
  • PCD (Project Completion Date)：計畫完成日期
  • PO Toolkit：計畫主持人工具包
  • Project Proposal：計畫提案
  • Project Session：計畫期

重要職務說明

1. 計畫主持人
  • 代表所屬經濟體負責申請與執行計畫，包括撰寫概念文件、計畫提案，提交監控報告、完工報告等。
  • 首次擔任者，須與計畫主任或祕書處計畫管理部門討論，確認計畫內容、時程等相關事宜。
  • 可依計畫難易度籌組委員會，由委員會擬定活動時程與地點、與會者、委託外包等相關事宜。
  • 所有相關文件請見計畫主持人工具包。
  • 其他事項請見 APEC 出版工具包（PDF）。
2. 計畫主任
  • 負責指導計畫主持人處理計畫相關事宜。
  • 收取並提交概念文件：第一階段提交至論壇 (包括工作小組、委員會) 執行內部審查，第二階段將最終版概念文件提交至 APEC 祕書處計畫管理部門。
  • 收取計畫提案並提交至 APEC 祕書處。
  • 收取並提交完整監控報告至相關論壇，摘要監控報告則提交至預算管理委員會。
  • 收取完工報告並提交至 APEC 祕書處。

必須提交的文件

1. 提交文件
  • 概念文件 (Concept Note)
  • 計畫提案 (Project Proposal)
  • 監控報告 (Monitoring Report)：舊版 (2018 年以前)、新版 (2019 年以後)
  • 完工報告 (Completion Report)：舊版 (2018 年以前)、新版 (2019 年以後)
2. 填寫上述文件時皆應參考：APEC 計畫指南 (Guidebook on APEC Project)
  • 目前為第 14 版。
  • 請查看第 14 版與先前版本差異。
3. 上述文件皆可取自計畫主持人工具包 (PO Toolkit)。
4. 建議詳讀 2019 年變更的內容，請見版本差異說明：簡要版、詳細版。
